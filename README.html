acrophony

An experimental React GUI for acrophonic alphabets.

Running it

npm install
npm start

Tests

npm install
npm test

Dependencies

  • acrophonic-alphabets: A collection of 92 Alpha-Bravo-Charlie-style alphabets from around the world.
  • alphabet: The english alphabet as a string or array, upper and lowercase.
  • domready: modern domready
  • jus: Squeeze delicious data out of a directory full of files
  • react: React is a JavaScript library for building user interfaces.
  • react-dom: React package for working with the DOM.

Dev Dependencies

  • enzyme: JavaScript Testing utilities for React
  • mocha: simple, flexible, fun test framework
  • react-addons-test-utils: This package provides the React TestUtils add-on.
